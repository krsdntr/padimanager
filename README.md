# PadiManager 🌾

**PadiManager** adalah aplikasi manajemen pertanian padi modern yang dirancang untuk membantu petani Indonesia mengelola siklus tanam secara lebih cerdas, efisien, dan terstruktur.

![PadiManager Banner](https://github.com/user-attachments/assets/0aa67016-6eaf-458a-adb2-6e31a0763ed6)

## ✨ Fitur Unggulan

- 📅 **Penjadwalan Pintar**: Buat jadwal tanam otomatis berdasarkan varietas padi (Ciherang, Inpari, Hipa, dll).
- 💰 **Manajemen Keuangan**: Catat HPP (Biaya Produksi), pendapatan panen, serta pantau ROI (Return on Investment) secara real-time.
- 📊 **Analisis Kinerja**: Grafik perbandingan antar siklus untuk mengevaluasi produktivitas dan profitabilitas.
- 📚 **Edukasi & Panduan**: Ensiklopedia varietas padi dan panduan pemupukan/pengairan (AWD).
- 📲 **Progressive Web App (PWA)**: Dapat diinstal di HP atau desktop dan bekerja 100% secara **OFFLINE**.
- 🔒 **Privasi Terjamin**: Seluruh data tersimpan secara lokal di perangkat Anda (Local-first).

## 🚀 Teknologi yang Digunakan

- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Database**: IndexedDB (Local Storage)
- **Icons**: Lucide React
- **Charts**: Recharts
- **Build Tool**: Vite

## 🛠️ Cara Menjalankan Lokal

1.  **Clone Repository**:
    ```bash
    git clone https://github.com/krsdntr/padimanager.git
    cd padimanager
    ```
2.  **Instal Dependensi**:
    ```bash
    npm install
    ```
3.  **Jalankan Development Server**:
    ```bash
    npm run dev
    ```
4.  **Akses Aplikasi**:
    Buka `http://localhost:3000/padimanager/` di browser Anda.

## 📦 Build untuk Produksi

Untuk membuat bundle produksi, jalankan:
```bash
npm run build
```
